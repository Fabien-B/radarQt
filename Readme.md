RadarQt
=======

A simple quick and dirty GUI to visualize lidar data.

:warning: This tool is really QUICK and DIRTY.

Launch it with `./radarqt.py /dev/ttyUSB0`.

You can test lidar without GUI: `./ld06.py /dev/ttyUSB0`.
